Built-in Functions

Shipping functions registered in src/polonio/runtime/builtins.cpp (v0.1.0).

String Helpers

len(value)

Converts the argument to a string (via the runtime output formatter) and returns its length in characters.

<% echo len(123) %> <!-- prints 3 -->

substr(text, start[, length])

Extracts a substring; start accepts negative values and length trims the slice.

<% echo substr("polonio", -3) %> <!-- nio -->

lower(text) / upper(text)

ASCII-only case conversion.

<% echo upper("hola") %>

trim(text)

Removes leading/trailing spaces, tabs, carriage returns, and newlines.

<% echo trim("  hi \n") %>

replace(text, from, to)

Replaces all occurrences of from with to.

<% echo replace("a-b-a", "-", "/") %>

split(text, sep)

Returns an array of strings split by sep (empty separators yield the whole string).

<% var parts = split("a,b,c", ",") %>

contains(haystack, needle)

Boolean substring search.

<% if contains(_SERVER["HTTP_HOST"], "beta") %>...end

starts_with(text, prefix) / ends_with(text, suffix)

Return true if the string begins/ends with the pattern.

<% if starts_with(slug, "docs/") %>

htmlspecialchars(text) / html_escape(text)

Both functions escape &, <, >, ", and '. Two names exist for compatibility; they produce identical output.

<% echo html_escape(_POST["message"]) %>
Use html_escape (or htmlspecialchars) before interpolating user input, because `$var` does not auto-escape.

Array Helpers

count(arrayOrObject)

Returns the length of an array or number of keys in an object.

<% echo count(users) %>

push(array, value)

Appends in-place and returns the new length.

<% push(todos, {"title": "Docs"}) %>

pop(array)

Removes and returns the last element (or null if empty).

<% last = pop(todos) %>

shift(array) / unshift(array, value)

shift removes the first element. unshift inserts at the beginning and returns the new length.

<% unshift(nav, "Home") %>

concat(a, b)

Creates and returns a new array containing all items from a followed by b.

<% all = concat(drafts, published) %>

join(array, sep)

Concatenates array elements into a string with the supplied separator.

<% echo join(tags, ", ") %>

slice(array, start[, length])

Returns a shallow slice. Negative starts count from the end; omitting length slices to the end.

<% recent = slice(posts, -3) %>

range(count)

Produces [0, 1, ..., count-1]. Non-positive counts return an empty array.

<% for i in range(3) %><li>$i</li><% end %>

Object Helpers

keys(object)

Returns a sorted array of keys.

<% for key in keys(params) %>...end

values(object)

Returns the object’s values sorted by key order.

<% vals = values(settings) %>

has_key(object, name)

Boolean membership check.

<% if has_key(_SERVER, "HTTP_HOST") %>

get(object, name[, default])

Returns object[name] or default (null by default) when the key is missing.

<% theme = get(_COOKIE, "theme", "dark") %>

set(object, name, value)

Mutates the object in place and returns the assigned value.

<% set(config, "locale", "es") %>

Math Helpers

abs(n), floor(n), ceil(n), round(n)

Standard numeric helpers (all arguments must be numbers).

<% echo floor(price) %>

pow(base, exp) / sqrt(n)

Exponentiation and square root (negative inputs to sqrt raise runtime errors).

<% echo pow(2, 8) %>

min(a, b) / max(a, b)

Return the smaller/larger of two numbers.

<% limit = min(requested, 100) %>

rand() / randint(min, max)

rand returns a floating-point value between 0 and 1. randint returns an integer in the inclusive range.

<% echo randint(1, 6) %>

Type Helpers

type(value)

Returns the runtime type name (null, bool, number, string, array, object, function).

<% echo type(user) %>

tostring(value) / to_string(value)

Serialize values exactly as the output buffer would (numbers keep precision, arrays print JSON-like).

<% raw = to_string(_GET) %>

to_number(value)

Converts booleans/strings/null to numbers. Strings are trimmed and parsed via std::stod; invalid text triggers a runtime error.

<% age = to_number(_POST["age"]) %>

is_null, is_bool, is_number, is_string, is_array, is_object, is_function

Return booleans for runtime type checks. is_function covers user and builtin functions.

<% if is_array(input) %>...end

Date Helpers

now()

Returns the current UNIX timestamp (seconds since epoch).

<% stamp = now() %>

date_format(epoch, pattern)

Formats seconds-since-epoch using tokens YYYY, MM, DD, HH, mm, SS.

<% echo date_format(now(), "YYYY-MM-DD") %>

date_parts(epoch)

Returns an object with year, month, day, hour, minute, second (UTC).

<% parts = date_parts(now()) %>

date_add_days(epoch, days)

Adds days (can be fractional) to the timestamp by multiplying by 86400.

<% tomorrow = date_add_days(now(), 1) %>

date_parse(text)

Parses YYYY-MM-DD or YYYY-MM-DD HH:MM:SS (space or T separator) and returns epoch seconds.

<% due = date_parse("2026-01-05") %>

Output Helpers

print(...values) / println(...values)

Write values to the template output buffer (identical to echo); println appends a newline.

<% println("Debug", version) %>

nl2br(text)

Converts \r\n, \n, and \r into HTML <br>\n pairs.

<% echo nl2br(_POST["bio"]) %>

debug(value)

Writes a single-line description of the value to stderr without affecting template output (handy inside CGI).

<% debug(_SERVER) %>

HTTP & Response Helpers

These functions require CGI mode (or any code path with an attached ResponseContext). Calling them without CGI will raise “CGI mode only”.

http_status(code) / status(code)

Sets the HTTP status line. Accepts integers from 100–599.

<% http_status(302) %>

http_header(name, value) / header(...)

Adds or overwrites headers. Call with either ("Header: value") or (name, value). Content-Type headers replace previous ones.

<% http_header("X-Powered-By", "Polonio") %>

http_content_type(value)

Convenience helper that sets Content-Type (and removes previous values).

<% http_content_type("application/json") %>

redirect(target[, status])

Sets a 302 (or custom 3xx) status plus the Location header.

<% redirect("/login") %>

urlencode(text) / urldecode(text)

Percent-encodes or decodes URL/query-string fragments using the classic application/x-www-form-urlencoded rules.

<% echo urlencode("hola mundo") %>
These helpers tie directly into the CGI response buffer, so headers are emitted before the template body when ResponseContext::emit runs.
